# JacqLaw — Law Offices of Jacquelyn R. Goncalves

Modern Next.js website for [jacqlaw.com](https://www.jacqlaw.com/) — an immigration law firm in Mystic, Connecticut serving Rhode Island and Connecticut.

## Pages

- **Home** — Hero, services, FAQs, testimonials, and CTAs
- **About** — Attorney bio, Navy service, concierge-style approach
- **FAQ** — All frequently asked questions
- **Contact** — Office info, map, and contact form
- **Book Now** — Consultation request form
- **Services** — Family-based immigration practice areas
